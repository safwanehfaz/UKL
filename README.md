# UKL - Unpacker Kitchen for Linux

UKL is a Linux-based fork of UKA (Unpacker Kitchen for Android) that provides comprehensive tools and scripts for Android firmware manipulation, including:

- **Unpacking and repacking** system, vendor, and boot images
- **Image format conversion** for `.img`, `.new.dat`, `.new.dat.br` files
- **Raw image mounting** capabilities
- **Multi-part image assembly** for Qfil firmware files
- **Payload.bin extraction** and processing
- **Boot/Recovery image repackaging**
- **File context extraction** from boot images
- **On-device file_contexts.bin conversion**

**Use Cases:**
- Android firmware extraction and modification
- Device customization and ROM development
- Firmware analysis and debugging
- Cross-platform compatibility with Linux systems

## Development status
UKL is currently in active development, with ongoing improvements and feature additions. The project is open-source and welcomes contributions from the community. For the latest updates, please refer to the GitHub repository.
